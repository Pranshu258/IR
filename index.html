REPRESENTATIVE SKYLINES WITH NOISY COMPARISONS

Skyline Queries on Databases

Given a set of data items, the skyline is the subset of items that are not "dominated". An item is dominated if there is another item that is superior for every criterion. Representative Skyline is set of items that best represent the skyline set itself. Now, in a dataset with noisy comparisons, the criterion values are not known. However, we have an oracle who can tell us which item is superior for a given criterion. Implemented an application that can compute Representative Skylines in a dataset with Noisy Comparisons.


C++

WAREHOUSE ORDER PICKING

Algorithms and Data Analysis

The order picking process is the process of retrieving products from specified storage locations in a warehouse on the basis of customer orders. It is in general one of the most time consuming processes in warehouses and contributes for a large extent to warehousing costs. We can reduce this cost by determining optimal order picking routes.

Designed and implemented a heuristic for finding good warehouse order picking paths. Also implemented a simulator that can be used to test, analyse and improve any order picking algorithm meant to be used for rectangular warehouses with similar constraints.


HTML5/CSS3, Javascript
