RENJU GAME


RENJU is a strategy board game played by two people. One of them takes the black stone and the other one white. They place their stones by turns on the board. The player who first makes five stones (of his/her colour) in a line wins the game. The line can be in any direction, horizontal, vertical or any of the two diagonals.
Visit https://en.wikipedia.org/wiki/Renju for more information.



PROJECT

An application that allows the user to play RENJU against the computer

TECHNOLOGIES USED

PYTHON/PYGAME, GIT


ALGORITHMS

MINIMAX

A minimax algorithm is a recursive algorithm for choosing the next move in an n-player game, usually a two-player game.
For more information visit:
https://en.wikipedia.org/wiki/Minimax

ALPHA-BETA PRUNING

Alpha-Beta pruning is a search algorithm that seeks to decrease the number of nodes that are evaluated by the minimax algorithm in its search tree.
For more information visit:
https://en.wikipedia.org/wiki/Alpha-beta_pruning
